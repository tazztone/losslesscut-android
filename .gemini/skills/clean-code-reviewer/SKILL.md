# Clean Code Reviewer
- **MVVM Architecture**: Maintain strict boundaries between View, ViewModel, and Model layers.
- **Naming Conventions**: Use descriptive, intention-revealing names.
- **SOLID Principles**: Focus on Single Responsibility for Engines and Utilities.
